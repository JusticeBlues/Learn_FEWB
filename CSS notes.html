CSS (Cascading Style Sheets) : The language that describes the visual appearance of webpages including properties such as layout, color and font

You can style pages in three ways:

  1. add the style attribute to individual elements on only the opening tag (i.e.: <p style="color: green"> *TEXT HERE* </p>)
    See? GREEN TEXT!
  2. Add the <style> opening and closing tags in the head of the HTML code followed by the element type curly brackets and the style attributes. Example below:
    <head>
    <style>
    p {color: green}
    </style>
    ... Check HTML for why this text is blue!
  3. Create a new text file with the file extension ".css"
    Use the html element <link rel="stylesheet" href="URL to CSS file">
Classes and ID's:

To add multiple styles to an element put a semicolon between attribute values (i.e.: p { color:blue; text-align:center;} )

To affect multiple elements of the same type but not all of that specific element use the html attribute class on the opening tag of an element and give it a name (i.e.: <p class="primary">) now any style change applied to "primary" will only affect primary class items not all p items.
to change the style of a Class put a . before the selector so instead of p { color:green;} for the primary class .primary { color:green; } only the elements labeled with the primary class attribute will be affected.

ID is used when you want to style a SINGULAR UNIQUE item on the page as opposed to a class which affects multiple elements.
To set an ID use the html attribute ID on the opening tag of an element and give it a name (i.e.: <div ID="unique">)
to change the style of an ID put a # before the selector so instead of div { border: 1px solid blue} which would affect all div elements instead type #unique { border: 1px solid blue;} only that singular item on the page will be affected.
DO NOT USE THE SAME ID MORE THAN ONCE!

You can combine selectors by typing the parent selector followed by the child selector before the declaration block (i.e.: li em { color:red;}) This will make all em elements in a list red.

1 px = 1/96" 1pt 1/72" 37.8px = 1 cm 1em = relative to font size of an element

Box example:

border: 1px solid green
margin: 1em
padding: 0.5em
width: 150px

<p style="border: 1px solid green; width: 150px; height:90px; padding: 0.5em; margin:1em">
