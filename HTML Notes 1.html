My HTML Notes :

Note to reader: the ampersand symbol followed by lt or gt and a semicolon creates the angle brackets visible on the webpage that do not trigger element tags. (i.e < >) using the actual symbols in a text editor software will instead trigger an element tag and it will not be visible in a browser.
  1. The web consists of HTML documents and other resources like images and videos interconnected through hyperlinks.
  2. Some terms:
    • Syntax - The basic rules of a language.
    • Formalism - Computers are literal.
    • Nesting - Bits of code inside other bits of code in an orderly manner.
    • https://developer.mozilla.org/
  3. Text is letters, numbers, symbols and spaces.
  4. Example of opening and closing tag respectively. <strong> </strong>
  5. To break a line type <br>, The br element is a void element which has no contents and no closing tag
  6. To make paragraphs type <p></p>
  7. Back to nesting... elements cannot overlap
    1. Incorrect: <em><strong>Hello<em><strong>
    2. Correct: <em><strong>Hello<strong><em>
  8. Headings for sections can be created using <h1></h1> with the 1 being replaced all the way down to 6 with decreasing size.
    See "My HTML Notes" for example of h1 tag
  9. Block Elements : i.e.: p and div elements create a block around the content within.
    Inline Element : i.e.: Tags like em or strong do not and just alter text as it flows normally.
  10. Always nest li tags inside of ol (ordered list) or ul (unordered list) tags.
  11. To change ordered list bullet <ol type="x">
    replace "x" with 1, A, a, I or i for their respective bullet styles.
  12. URL: Uniform Resource Locator
    1. Uniform = Standardized
    2. Resource = A file or other data object
    3. Locator = Address for finding something
  13. Hyperlinks can be created by using the tag <a href="Qualified URL">"Hyperlink clickable text"</a>
    you can also put "(spacebar)target=_blank" after the quoted Qualified URL to make the linked resource open in a new tab.
  14. Images can be inserted by using the tag <img src="Qualified Image URL" alt="Description of Image (i.e: dog photo)"> there is no closing tag.
    If the image you want to use is stored locally in the same directory as the HTML file you can use a relative URL like "Dog.jpg" rather than a qualified URL which is beneficial if you are uploading or sending the HTML file with the image file.
  15. Relative links can be used for anchor hypertext references too so as long as the connected linked resource is in the same directory as the HTML document.
  16. For placeholder images use the qualified URLs from placebear or placekitten (i.e: https://placebear.com/200/300 you can change the 200 and 300 for any width or height respectively)
  17. When you've finished writing your HTML use https://validator.w3.org to ensure that your HTML is written correctly.
